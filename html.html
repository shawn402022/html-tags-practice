Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the 'href' attreibute. Go to DuckDuckGo to search for more information.

Anchor tags can also create links to a new location on the same page. Instead of putting a URL goes in the "href" attribute, just use the "id" of the element you want to navigate to. This allows you to Jump to the top of the page.

Setting 'target="_blank" `will open the link in a new window. See MDN for moew details

Text Emphasis

Italic

The 'em element adds extra emphjasios to the text within the opening and closing tags. This shows up as italic in most browsers

Bold

You can also use the 'strong' element to emphasize text. This shows up as bold text in most browsers.
